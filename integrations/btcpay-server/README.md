# <img src="logo.png" height="20"> Btcpay Server

Accept and process cryptocurrency payments through a self-hosted, non-custodial payment processor. Create and manage invoices with configurable amounts, currencies, and expiration. Monitor invoice status changes (New, Processing, Expired, Settled, Invalid) and receive webhook notifications for payment events. Manage on-chain Bitcoin wallets including viewing balances, generating addresses, and creating transactions. Handle Lightning Network payments, create Lightning invoices, and check node balances. Create pull payments and manage payouts with approval workflows for both on-chain and Lightning. Set up payment requests for freelancers or bill pay. Configure and manage multiple stores, each with independent payment methods, wallets, and user access controls. Create Point of Sale and Crowdfunding apps. Issue full or partial refunds on invoices. Manage users, API keys, server settings, notifications, and reporting.
