# Celigo

Manage integration flows between cloud applications on the Celigo (integrator.io) iPaaS platform. Create, update, delete, and invoke integration flows that export data from one application and import it into another. Manage connections with credentials for integrated apps, and test or execute virtual exports/imports. Handle errors programmatically by retrieving, retrying, and resolving flow errors. Organize flows and connections within integrations, and create exportable integration templates. Build custom API endpoints with low-code tools, store arbitrary state data between flow executions, manage users and permissions, configure webhook listeners to trigger flows in real-time, and manage lookup caches and file definitions for EDI parsing.
