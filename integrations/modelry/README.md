# <img src="logo.png" height="20"> Modelry

Manage 3D product visualizations for eCommerce. Create, retrieve, and delete products with SKUs, dimensions, tags, and external URLs. Submit and track modeling requests for 3D model creation. Manage embeddable 3D and AR viewers for product pages. Handle product assets across multiple 3D file formats (glTF, GLB, USDZ, and more). Organize work across workspaces.
