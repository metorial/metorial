# <img src="logo.png" height="20"> Resend

Send, receive, and manage emails programmatically. Send transactional and marketing emails with HTML, plain text, templates, attachments, scheduling, and batch sending. Manage sent and received email metadata, including attachment downloads through Slate attachments. Manage sending domains with DNS verification (DKIM, SPF, DMARC) and open/click tracking. Create and organize contacts with contact properties, segments, and topic-based subscription preferences. Send broadcast campaigns immediately or on a schedule with automatic unsubscribe handling. Create, publish, duplicate, and manage reusable email templates. Manage automations, events, API keys, webhook endpoints, and account logs for real-time email, contact, and domain event visibility.
