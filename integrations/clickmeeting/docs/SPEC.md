# Slates Specification for Clickmeeting

## Overview

ClickMeeting is a communication platform that offers the ability to create and manage virtual meetings, conference rooms and teleconference centers. Its API allows programmatic management of webinars, meetings, attendee registrations, session data, recordings, and file storage.

## Authentication

ClickMeeting uses **API Key** authentication.

- Every API request must be authenticated by an API key, which can be generated by logging in to a ClickMeeting users account and navigating to the Settings tab.
- The API key will be transmitted as a request's header with every request made. Name of the header is 'X-Api-Key'.
- To generate a key: go to your ClickMeeting Account Settings in the upper right corner of the screen and open the API tab. Next, click the Add new button, enter API key name and click Save. You can now copy your API key. Be sure to copy it as soon as you generate it, as you would not be able to once you leave/refresh the page.
- Base URL: `https://api.clickmeeting.com/v1/`
- For security reasons, the API is available under SSL certificate.

**Example request header:**

```
X-Api-Key: YOUR_API_KEY
```

## Features

### Conference/Room Management

Create, update, retrieve, and delete conference rooms (meetings and webinars). There are two types of rooms: time scheduled and permanent. Rooms can be configured with access types (open, password-protected, or token-protected), lobby settings, timezone, registration options, and advanced settings like recorder autostart, phone gateway, end-to-end encryption, and custom branding skins.

- Conference rooms that are inactive for more than 6 months will be removed from the list automatically and stored in a ClickMeeting database.
- The number of each room type available for creation depends on the type of ClickMeeting account you have.

### Access Tokens

If the access type is set to token-protected for a specific event, you can generate access tokens for meeting or webinar participants. Access token usage could be a form for paid access to your meetings or webinars. You can sell the access token via CRM or email marketing platforms.

- Tokens can be created individually or in bulk, retrieved, and deleted for a given conference.

### Attendee Registration

ClickMeeting allows attendees registration; turn on registration if you would like participants to register before attending the conference. Using API you are able to register new attendees, receive list of all registered participants.

- Registrations can be retrieved by status (active, pending) for a conference or for a specific session.

### Session & Attendee Data

Retrieve session history for a conference, including details about individual sessions and their attendees. You can generate PDF reports for sessions in various languages.

- Session data includes attendee-level information useful for analytics and follow-up.

### Invitations

Send email invitations for a conference room to specified recipients. The invitation language can be configured per request.

### Auto-login URLs

Generate auto-login hash URLs for conference rooms, allowing participants to join directly without manually entering credentials. Role-based hashes are available (host, presenter, listener).

### File Library (Storage)

Upload, download, list, and delete files in your account's file library. Files can also be associated with specific conference rooms. Recordings from sessions can be retrieved and managed (listed or deleted) per conference.

### Contacts

Add contacts to your ClickMeeting address book for use in invitations and event management.

### Utility Data

Retrieve available timezone lists (optionally filtered by country) and phone gateway information for dial-in conference access.

### Chat Logs

Retrieve chat session logs, including the ability to download full chat content for a specific session.

## Events

The provider does not support events. ClickMeeting's API does not offer webhooks, callback URLs, or any built-in event subscription mechanism for real-time notifications.
