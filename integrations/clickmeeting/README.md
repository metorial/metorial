# <img src="logo.svg" height="20"> Clickmeeting

Create, manage, and delete virtual conference rooms (webinars and meetings) with configurable access types, registration, and advanced settings. Send email invitations to participants and generate auto-login URLs for direct room access. Manage attendee registrations, retrieve session history and attendee data, and generate PDF session reports. Create and manage access tokens for token-protected events. Upload, download, list, and delete files in the file library, and manage session recordings. Retrieve chat logs, manage contacts in the address book, and access utility data like timezones and phone gateways.
