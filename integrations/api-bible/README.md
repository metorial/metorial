# <img src="logo.png" height="20"> Api.bible

Access Bible scripture content in text and audio formats across nearly 2500 versions and over 1600 languages. Browse and filter Bible versions by language and metadata. Retrieve scripture content hierarchically by book, chapter, verse, passage, or section, returned in unified HTML format. Search Bible text by keyword or reference, scoped to specific books or ranges. List and retrieve audio Bible versions. Track scripture engagement via the Fair Use Management System (FUMS).
