# Feathery

Create, manage, and collect submissions for forms and data intake workflows. Build forms from templates, track user submissions, and export them as PDFs. Extract structured data from uploaded documents using AI. Fill and sign document templates (PDF, DOCX, XLSX) with field values and route them for e-signatures. Manage users and their form session progress. Create white-label workspaces with custom branding and feature limits. Invite team members with role-based permissions. Access logs for API errors, emails, and integrations. Configure webhooks for form completion, data received, file upload, and document signed events.
