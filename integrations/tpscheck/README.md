# <img src="logo.png" height="20"> Tpscheck

Verify UK phone numbers against the Telephone Preference Service (TPS) and Corporate Telephone Preference Service (CTPS) registers. Check single or bulk batches of up to 100 numbers for TPS/CTPS registration status, line type, carrier, location, and reachability. Retrieve compliance risk scores (0–100) for each number. Monitor numbers with automated 28-day re-checks to maintain ICO compliance. Access audit logs and exportable compliance reports. Check API credit usage and remaining allowances. Receive webhook notifications for bulk job completion, TPS status changes, low credits, and usage limits.
