# <img src="logo.svg" height="20"> Enginemailer

Manage email marketing campaigns, subscribers, and transactional emails. Add, update, deactivate, or delete subscribers with custom fields and tags. Create and send email campaigns with HTML content. Send transactional emails with personalization, attachments, and templates. Set up autoresponders and automated email sequences. Track campaign performance including open rates, click-through rates, bounces, and unsubscribes. Receive real-time webhook notifications for delivery, bounce, open, click, unsubscribe, and spam complaint events.
