# <img src="logo.jpeg" height="20"> Payhip

Manage coupons and software license keys for a Payhip e-commerce store. Create, list, and retrieve discount coupons with configurable options like percentage or fixed-amount discounts, usage limits, and date ranges. Verify, enable, disable, and track usage of software license keys tied to digital products. Receive webhook notifications for completed payments, refunds, subscription creations, and subscription cancellations.
