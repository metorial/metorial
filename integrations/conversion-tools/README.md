# Conversion Tools

Convert files between 100+ formats including documents (PDF, Word, Excel, PowerPoint), images (JPG, PNG, WebP, AVIF, HEIC), data formats (JSON, XML, CSV, YAML, Parquet), eBooks (ePUB, MOBI), audio/video (MP4, MP3, WAV), and subtitles (SRT, VTT). Extract text from images and scanned PDFs using OCR with 100+ language support. Capture websites as PDF or image screenshots with configurable viewport and device emulation. Transform structured data between JSON, XML, CSV, Excel, YAML, Parquet, JSONL, and BSON. Use AI-powered extraction to convert PDFs and images into structured JSON, Markdown, CSV, or Excel. Translate subtitles using AI across 30+ languages. Convert Markdown to PDF, HTML, ePUB, Word, or LaTeX. Remove EXIF metadata from images. Validate XML/XSD and format JSON. Conversion tasks run asynchronously with polling or webhook callback support for completion notifications.
