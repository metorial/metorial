# Vestaboard

Send and read messages on Vestaboard smart display devices. Compose messages as plain text or character code arrays with support for colored blocks and precise positioning. Format messages using VBML (Vestaboard Markup Language) with layout styling, templates, and absolute positioning. Configure transition effects (classic, wave, drift, curtain) between messages. Manage subscriptions to send messages across multiple boards. Supports Flagship (6×22), Vestaboard Note (3×15), and Note Array configurations via cloud, subscription, or local network APIs.
