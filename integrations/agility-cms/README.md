# <img src="logo.png" height="20"> Agility CMS

Manage and deliver structured content, pages, and media assets in a headless CMS. Fetch content items and lists with filtering and sorting, retrieve pages and sitemaps, and query content via GraphQL. Create, update, delete, publish, and unpublish content items and pages. Manage content workflows including approval requests, approvals, and declines. Upload, retrieve, and delete media assets and organize them in galleries and folders. Define and manage content models and schemas. Sync content incrementally using sync tokens for efficient caching. Manage URL redirections, instance users, and webhooks. Receive webhook notifications for content publish, save, and workflow events.
