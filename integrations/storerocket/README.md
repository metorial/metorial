# <img src="logo.svg" height="20"> Storerocket

Retrieve store locator account information, manage users, and access physical store location data. Get authenticated account details and plan information, list users with roles and permissions, and query store locations including addresses, phone numbers, operating hours, and custom fields. Supports filtering locations by geographic proximity to match customers with nearby stores. Provides a health check endpoint to verify API connectivity.
