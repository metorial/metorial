# Dreamstudio

Generate images from text prompts using Stable Diffusion models (Ultra, Core, SD 3.5). Edit images with inpainting, erasing, outpainting, search-and-replace, recoloring, background removal, and relighting. Upscale images using conservative, creative, or fast methods. Control image generation with sketch, structure, and style transfer tools. Generate video from text or images. Create 3D meshes and assets from 2D images. Check account credit balance.
