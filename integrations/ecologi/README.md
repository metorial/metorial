# Ecologi

Purchase trees and carbon avoidance credits to fund climate action projects through Ecologi. Retrieve impact statistics including total trees planted, CO₂e avoided, carbon removal, and habitat restoration for any Ecologi user profile. Supports test mode for simulating transactions without charges. Impact data is publicly accessible by username without authentication.
