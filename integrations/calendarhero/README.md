# <img src="logo.png" height="20"> Calendar Hero

Schedule and manage meetings with automated booking for 1:1 and group meetings. Retrieve and manage meeting types, query scheduled meetings by date range, and create meeting requests that find optimal times based on availability. Create and search contacts with detailed insights including job history, personality, and social media profiles. Subscribe to webhooks for meeting lifecycle events such as scheduled, created, expired, and cancelled meeting requests, as well as new contacts added via scheduling links.
