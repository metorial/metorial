# <img src="logo.svg" height="20"> Parallel

Perform AI-optimized web searches, extract content from URLs, run deep multi-step research tasks, and discover entities matching natural language criteria. Provides web-grounded chat completions compatible with OpenAI client libraries. Continuously monitor the web for new information on a configurable schedule and receive webhook notifications when relevant changes are detected. Supports structured JSON output schemas, multiple processing tiers for cost/thoroughness trade-offs, and asynchronous task execution with polling or webhook-based completion tracking.
