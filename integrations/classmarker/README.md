# Classmarker

Retrieve exam and quiz results, manage questions and categories, and control exam access for online assessments. Read test scores, user responses, and grading details. Add, edit, and organize questions with categories and point values. Manage access lists to control who can take exams. Discover available groups, links, and tests. Receive real-time exam completion results via webhooks.
