# <img src="logo.png" height="20"> Better Stack

Monitor uptime of websites, APIs, and services using HTTP, ping, TCP, and other check types. Create and manage incidents, including acknowledging, resolving, and escalating them. Configure on-call schedules, escalation policies, and rotations. Create and manage public status pages with custom branding and subscriber notifications. Track heartbeats for CRON jobs and scheduled workers. Manage telemetry log sources, custom metrics, and dashboards. Set up threshold, relative, and anomaly detection alerts. Query logs and metrics using ClickHouse-compatible SQL. Configure incoming webhooks to create or resolve incidents from external systems. Manage integrations with third-party observability tools. Receive webhook notifications for incident, monitor, and on-call events.
