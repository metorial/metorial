# <img src="logo.svg" height="20"> Gleap

Manage customer feedback, bug reports, and support tickets. Create, update, query, and delete tickets with filtering by status, priority, and assignee. Manage user sessions and contacts with custom data attributes. Add messages, agent replies, and internal notes to tickets with file attachments. Create and manage help center collections and articles. Run engagement campaigns including banners, emails, surveys, product tours, tooltips, checklists, modals, push notifications, and WhatsApp messages. Manage AI-powered support content for automated responses. Access project analytics and statistics. Configure message templates, team members, and project settings. Receive webhooks when feedback is created, updated, or deleted.
