# <img src="logo.jpeg" height="20"> Canvas

Manage courses, assignments, enrollments, and grading in Canvas LMS. Create and organize course content including modules, pages, files, quizzes, and discussions. Enroll users with specific roles, submit and grade assignments, manage rubrics, and track learning outcomes. Send messages between users, schedule calendar events, and configure appointment groups. Upload and organize files, manage groups, and handle SIS data imports. Access course and account-level analytics including participation data and student performance. Configure external tool integrations (LTI), perform content migrations, and administer account settings, roles, and permissions. Subscribe to real-time live events for actions like submissions, grade changes, enrollments, logins, and content updates.
