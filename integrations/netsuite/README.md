# <img src="logo.svg" height="20"> Netsuite

Manage Oracle NetSuite ERP data and operations. Create, read, update, and delete business records including customers, vendors, sales orders, purchase orders, invoices, inventory items, journal entries, and employees. Query data flexibly using SuiteQL (SQL-based querying) with JOINs, filtering, and aggregation. Manage the full financial lifecycle: general ledger entries, accounts payable/receivable, invoices, bills, payments, credit memos, and budgets. Handle order and inventory management including item fulfillments, inventory adjustments, transfer orders, and return authorizations. Transform records between types (e.g., sales order to invoice). Execute record actions such as approving transactions. Upload, download, and manage files in the File Cabinet. Access custom record types, custom fields, and custom RESTlet endpoints. Supports asynchronous processing for long-running operations.
