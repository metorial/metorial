# Codereadr

Manage barcode scanning workflows, validation databases, and scan records. Create and configure scanning services (record scans, validate scans, check in/out). Manage app users and their service access. Build and populate validation databases with barcode values and response text. Retrieve and filter scan records by service, user, date, or barcode value. Create data collection questions (text, dropdowns, photos, signatures, GPS) attached to scans. Generate QR codes and PDF417 barcodes from database values. Configure postback URLs for real-time scan forwarding and validation. Manage devices and kiosk mode settings.
