# <img src="logo.png" height="20"> Cutt.ly

Shorten URLs, manage shortened links, and retrieve click analytics. Create short links with custom aliases, branded domains, and UTM parameters. Edit link properties including destination URL, title, tags, password protection, expiration rules, mobile redirects, deep links, and AB/C split testing. Retrieve detailed click statistics with breakdowns by referrer, geography, device, browser, OS, and language. Supports team-based link management.
