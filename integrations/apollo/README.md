# <img src="https://provider-logos.metorial-cdn.com/apollo.jpg" height="20"> Apolloio

Search and prospect B2B contacts and companies from Apollo's people and organization database. Enrich people and organizations, manage contacts, accounts, deals, tasks, sequences, and call records, and query Apollo usage, conversation, and analytics APIs.

## Tools

### Search People

Search Apollo's API-optimized people database. Returns professional details and organization context, but not email addresses or phone numbers.

### Search Organizations

Search Apollo's organization database by name, domains, locations, headcount, technologies, funding/job signals, and related filters.

### Get Organization

Retrieve complete Apollo organization details by organization ID.

### Enrich Organization

Enrich one organization by domain, or up to 10 organizations by domain.

### List Organization Job Postings

List current job postings for an Apollo organization ID.

### Enrich Person

Enrich a person's profile data using Apollo's enrichment engine. Provide identifiers like email, name + company, or LinkedIn URL to get detailed profile information including employment history, contact details, and organization data. Supports both single and bulk enrichment (up to 10 records per request).

### Search Contacts

Search contacts that have been added to your Apollo account.

### Create Contact

Create a contact in your Apollo account.

### Update Contact

Update an existing contact.

### Bulk Create Contacts

Create up to 100 contacts in one request.

### Bulk Update Contacts

Update multiple contacts in one request.

### Update Contact Stages

Update the stage for one or more contacts.

### Update Contact Owners

Assign one or more contacts to a different owner.

### Search Accounts

Search for accounts (companies) that have been added to your Apollo database. Returns accounts your team has explicitly added — use Search Organizations for the broader Apollo company database.

### Get Account

Retrieve an Apollo account by account ID.

### Create Account

Create an account in your Apollo database.

### Update Account

Update an existing account.

### Bulk Create Accounts

Create up to 100 accounts in one request.

### Bulk Update Accounts

Update multiple accounts in one request.

### Update Account Owners

Assign one or more accounts to a different owner.

### List Deals

List all deals (opportunities) in your Apollo account. Returns deal details including name, amount, stage, owner, and associated account.

### Get Deal

Retrieve a specific deal by ID.

### Create Deal

Create a new deal.

### Update Deal

Update an existing deal.

### Search Sequences

Search for email sequences in your Apollo account. Returns sequence names, step counts, and activity status. Use this to find a sequence before adding contacts to it.

### Add Contacts to Sequence

Add one or more contacts to an Apollo sequence.

### Update Contact Sequence Status

Mark contacts in a sequence as finished or active.

### Search Tasks

Search for tasks created by your team in Apollo. Tasks track upcoming actions like emailing or calling contacts.

### Create Task

Create one or more Apollo tasks.

### Search Calls

Search Apollo call records.

### Create Call

Create a call record in Apollo for calls made in external systems.

### Update Call

Update an Apollo call record.

### Search Conversations

Search Apollo conversations by type, account, contacts, tags, trackers, organizations, date range, and scorecard filters.

### Get Conversation

Retrieve full details for a single Apollo conversation.

### Export Conversations

Request an asynchronous conversation export for a time range.

### Query Analytics Report

Query Apollo's analytics report engine.

### List Users

Retrieve all users in your Apollo organization.

### List Stages

Retrieve contact, account, or deal stages.

### List Email Accounts

Retrieve linked email inboxes for Apollo teammates.

### Get Usage Stats

Retrieve Apollo API usage statistics and rate limits.
