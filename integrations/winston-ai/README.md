# <img src="logo.svg" height="20"> Winston Ai

Detect AI-generated text and images, check for plagiarism, verify factual claims, and compare texts for similarity. Submit content as plain text, file URLs, or website URLs and receive confidence scores. AI text detection returns human scores (0–100), per-sentence scoring, readability analysis, and obfuscation attack detection across 12 languages. AI image detection identifies synthetic images, deepfakes, and extracts C2PA provenance and EXIF metadata. Plagiarism detection scans text against internet sources, returning matched sources with details across 45 languages. Fact checking extracts claims from content and verifies them against trusted sources with per-claim verdicts and references. Text comparison analyzes two texts for similarity with detailed match positions.
