## Overview

Winston AI is a content integrity platform that detects AI-generated text and images, checks for plagiarism, verifies factual claims, and compares texts for similarity. Its AI text detection supports English, French, Spanish, Portuguese, Dutch, German, Polish, Italian, Romanian, Indonesian, Tagalog, and Chinese Simplified. The image detection system identifies images generated by major AI models including ChatGPT Images, Midjourney, Stable Diffusion, Adobe Firefly, Ideogram, Leonardo, Runway, and others.

## Authentication

All API endpoints are authenticated using Bearer tokens. Developers work at dev.gowinston.ai, which lets you create API keys so your own software can call the detection engine automatically.

To authenticate, generate an API token from the [Winston AI Developer Dashboard](https://dev.gowinston.ai). Then include it as a Bearer token in the `Authorization` header of each request:

```
Authorization: Bearer <your-api-token>
```

Accounts are separate on purpose: your web-app username and password won't open the developer portal, and an API key won't sign you in to the app. The developer account is distinct from the consumer web app account at app.gowinston.ai.

No OAuth flow, scopes, or additional credentials are required — only the single Bearer token.

## Features

### AI Text Detection

The Human Score is a metric used by Winston AI to estimate the likelihood that a given piece of content was generated by an AI tool versus being written by a human. Content can be submitted as raw text, a publicly accessible file URL (.pdf, .doc, .docx), or a website URL. The API returns an overall score (0–100), optional per-sentence scores, a readability score, and detection of obfuscation attacks (zero-width spaces, homoglyph attacks).

- **Input options**: Plain text (minimum 300, maximum 50,000 characters), file URL, or website URL.
- **Language**: Can be set explicitly or auto-detected. Supports 12 languages for AI detection.
- **Sentence-level scoring**: Optionally returns individual sentence scores by setting the `sentences` parameter to `true`.
- **Model version**: A `version` parameter allows selecting the detection model version.

### AI Image Detection

Determines whether an image was generated by AI or created by a human. The API identifies manipulated faces, identity swaps, synthetic personas, and frame-extracted deepfakes. It also extracts C2PA content provenance metadata, EXIF/IPTC metadata, and detects AI watermarks.

- **Input**: A publicly accessible image URL (JPG, JPEG, PNG, or WEBP). Minimum resolution of 256×256 pixels.
- **Model version**: Configurable via the `version` parameter (options: 1, 2, 3, or "latest").
- **Output**: Human score (0–100), AI/human probability, C2PA metadata, EXIF data, AI watermark detection, and image technical metadata.

### Plagiarism Detection

Scans text against internet sources to identify duplicate or similar content. Returns an overall plagiarism score, matched sources with details, and identifies identical vs. similar word matches.

- **Input options**: Plain text (100–120,000 characters), file URL, or website URL.
- **Language**: Supports 45 languages with auto-detection.
- **Country**: A country code can be provided to tailor search results.
- **Excluded sources**: An array of domains or URLs to exclude from the scan.
- **Output**: Overall score, list of matching sources (with URLs, titles, word counts, and plagiarism sequences), citation detection, and obfuscation attack detection.

### Fact Checking

Verifies the factual accuracy of content by extracting claims and checking them against trusted sources. Returns per-claim verdicts (e.g., "SUPPORTED") with explanations and supporting links.

- **Input options**: Plain text (300–10,000 characters), file URL, or website URL.
- **Language**: Supports 38 languages with auto-detection.
- **Output**: Overall accuracy score (0–100), list of extracted claims (up to 12 per request) with individual verdicts, confidence scores, explanations, and reference links.

### Text Comparison

Compares two pieces of text to identify similarities and differences, returning a detailed similarity analysis.

- **Input**: Two text strings, each up to 120,000 characters.
- **Output**: Overall similarity score (0–100), per-text matching word counts, similarity percentages, and detailed item-level match positions.

## Events

The provider does not support events. Winston AI's API is request-response only and does not offer webhooks, event subscriptions, or purpose-built polling mechanisms.
