# <img src="logo.png" height="20"> Cardly

Send personalized, handwritten greeting cards, postcards, and letters at scale via physical mail. Place orders with customizable artwork, message templates, and recipient details for delivery to 50+ countries. Manage card artwork by uploading page images, and use message templates with variable substitution for personalization. Create and manage contact lists with custom fields, sync contacts via upsert, and configure automation rules for triggers like birthdays. Generate preview PDFs before placing live orders. Browse available media types, handwriting fonts, writing styles, and decorative doodles. Query account credit balances and transaction history. Manage organization users and invitations with configurable permissions. Set up webhooks to receive notifications for order, shipping, gift card redemption, QR code scan, and delivery events.
