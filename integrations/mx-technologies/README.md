Aggregate, enhance, and verify financial data by connecting to thousands of financial institutions. Create and manage users and their financial institution connections (members). Aggregate account balances, transactions, and account details across connected institutions. Verify accounts instantly (IAV) with account and routing number verification. Retrieve account owner identity information for KYC processes. Perform balance checks on connected accounts. Access extended transaction history for underwriting and analysis. Retrieve account statements in PDF format. Enhance and categorize raw transaction data with merchant information. Verify accounts via microdeposit flows. Generate personalized financial insights and recommendations. Retrieve enhanced investment holdings data. Search and browse supported financial institutions and their capabilities. Track data changes across all users via the Reporting API. Receive webhooks for aggregation, balance, verification, connection status, transaction, and other data change events.
