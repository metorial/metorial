# <img src="logo.jpeg" height="20"> Parseur

Extract structured data from emails, PDFs, and other documents using AI-powered parsing. Upload and process documents (PDFs, images, emails), manage mailboxes (parsers) configured for specific document types, and retrieve parsed data in JSON, CSV, or XLS formats. Create and manage templates for data extraction, configure webhooks for event-driven notifications (e.g., document processed, process failed), and access mailbox schemas. Supports ready-made templates for invoices, resumes, leads, travel bookings, and more.
