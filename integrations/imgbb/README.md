# Img Bb

Upload images to ImgBB and receive publicly accessible hosted URLs. Supports uploading via binary file, base64-encoded data, or image URL. Set optional auto-expiration times for temporary hosting. Returns multiple image variants including original, thumbnail, and medium-sized versions with direct, viewer, display, and delete URLs.
