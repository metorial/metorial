# <img src="logo.svg" height="20"> Ynab

Manage personal budgets, accounts, transactions, and financial categories. Create, read, update, and delete transactions including split transactions and scheduled recurring transactions. Track account balances across checking, savings, credit cards, and loans. Manage budget categories with goal tracking, view monthly budget summaries including income, activity, and "Ready to Assign" amounts. Import bank transactions with automatic deduplication and matching. Retrieve payee information and locations. Export full budget data and efficiently sync changes using delta requests.
