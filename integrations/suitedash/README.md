# Suitedash

Create, retrieve, update, and list contacts and companies in the CRM with full custom field support. Manage projects and tasks, including creation, updates, and search. Subscribe contacts to marketing audiences for email campaigns. Retrieve metadata schemas for custom fields and marketing audiences to enable two-way data sync. Receive webhooks for subscription lifecycle events (created, canceled, expired, unpaid), project changes (created, updated, deleted), and custom automation-triggered events.
