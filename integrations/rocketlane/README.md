# <img src="logo.png" height="20"> Rocketlane

Manage customer onboarding and implementation projects. Create, update, archive, and delete projects with phases, tasks, and custom fields. Track time entries, manage resource allocations, and handle financial data like invoices. Create and manage collaborative spaces and documents for sharing with customers. Manage companies (customers), users, conversations, and comments. Configure project templates, custom fields for projects/tasks/companies, and receive webhook notifications for task events.
