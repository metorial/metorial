# <img src="logo.png" height="20"> 2 Chat

Send and receive WhatsApp messages (text, images, audio, video, PDFs) to individuals and groups. Manage WhatsApp contacts, groups, catalogs, and connected numbers. Verify WhatsApp phone numbers, set profile status and pictures, and generate QR codes for white-label WhatsApp connections. Manage virtual phone numbers and voice calls across 90+ countries. Subscribe to webhooks for real-time events including message delivery receipts, group membership changes, call status updates, and order notifications.
