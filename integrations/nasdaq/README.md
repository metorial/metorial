Access financial market data including real-time, delayed, and historical pricing for equities, options, futures, FX, and economics. Retrieve end-of-day stock prices, company fundamentals, earnings estimates, analyst ratings, and options implied volatility. Query historical and reference datasets with filtering by date range, columns, and row-level criteria. Stream real-time market data feeds including quotes, trades, order book depth, and index values. Search and browse available datasets, retrieve dataset metadata, and perform bulk data downloads. Data is available in CSV, JSON, and XML formats.
