# <img src="logo.png" height="20"> Mailboxlayer

Validate and verify email addresses in real time. Check email syntax against RFC standards, verify existence via MX record lookup and SMTP checks, detect disposable and free email providers, identify role-based addresses, and receive a deliverability quality score. Supports single and bulk email validation, typo detection with "did you mean" suggestions, and catch-all detection.
