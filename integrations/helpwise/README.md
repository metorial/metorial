# Helpwise

Manage shared inboxes and customer conversations across email, SMS, WhatsApp, live chat, and social media channels. Send and list emails, create and manage mailboxes, track and assign conversations, add internal notes, manage contacts, upload attachments, organize with tags, and use email templates. Manage teams, users, and webhooks for real-time event notifications on conversation lifecycle, assignment, and message activity.
