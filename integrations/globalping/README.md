# Globalping

Run network diagnostic measurements from globally distributed probes. Create ping, traceroute, DNS, MTR, and HTTP tests targeting any public domain or IP address. Select probe locations by country, continent, city, US state, region, ASN, ISP, or cloud region. Retrieve measurement results by polling with a measurement ID. List and filter available online probes by location or network. Check API health and availability. Useful for monitoring latency, debugging routing issues, optimizing anycast networks, and detecting censorship across different geographies.
