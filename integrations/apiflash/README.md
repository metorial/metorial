# Apiflash

Capture screenshots of any publicly accessible URL using a Chrome-based rendering engine. Configure viewport dimensions, capture full-page scrollable content, or target specific HTML elements via CSS selectors. Supports PNG, JPEG, and WebP output formats. Inject custom CSS and JavaScript to modify page appearance before capture. Pass custom headers, cookies, and proxies to access authenticated or geo-restricted pages. Hide cookie banners automatically. Export screenshots directly to Amazon S3 buckets. Control caching behavior and force fresh captures. Extract HTML content from target pages. Monitor API quota usage and remaining credits.
