# <img src="logo.jpeg" height="20"> Landbot

Manage chatbot conversations and customers across web, WhatsApp, Messenger, and API channels. Send text messages, images, and location links to customers. Send WhatsApp message templates for outbound campaigns. Retrieve and update customer profiles, assign customers to bots or agents, and manage WhatsApp opt-ins. Create, update, and track support tickets. Assign customers to specific bot flows and direct them to particular steps. Manage agents, channels, and channel configurations. Receive webhook notifications for message events (button clicks, text, media) and bot events (agent assign/unassign).
