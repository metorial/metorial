# <img src="logo.svg" height="20"> Docsumo

Upload, process, and extract structured data from documents using AI-powered models. Supports invoices, bank statements, tax forms, insurance documents, and more. Retrieve extracted key-value pairs and table data, review document status, generate review URLs, inspect account usage and document types, retrieve bank statement analytics, manage cases and workflows, and receive webhook notifications on document status changes.
