# <img src="logo.png" height="20"> Bunny CDN

Manage CDN pull zones, cache purging, edge storage, video streaming, DNS, and edge computing on bunny.net. Create and configure pull zones with caching rules, custom hostnames, SSL, geo-blocking, and token authentication. Upload, download, list, and delete files in edge storage zones. Host and stream videos with transcoding, resumable uploads, DRM, captions, and playback security. Create and manage DNS zones and records. Configure WAF rules, DDoS protection, bot detection, and rate limiting via Bunny Shield. Deploy serverless edge scripts and containerized applications. Access bandwidth, request, and cache hit statistics. Receive webhooks for video processing status changes including encoding progress, completion, and caption generation.
