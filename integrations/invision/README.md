# in Vision

InVision was a digital product design platform that provided design collaboration, prototyping, and design system management tools. It was permanently shut down on December 31, 2024, and no API integration is currently possible. Prior to shutdown, its limited API capabilities included: retrieving design tokens (colors, typography, spacing) and icons from Design System Manager (DSM), and SCIM-based user provisioning for Enterprise accounts. InVision never offered a general-purpose API for managing prototypes, projects, or core platform resources. Its Freehand collaboration tool was acquired by Miro in 2023.
