# <img src="logo.png" height="20"> Botbaba

Manage and deploy no-code chatbots for WhatsApp and web channels. Retrieve bot listings, send WhatsApp template messages via 360Dialog or Gupshup, and handle e-commerce workflows including ordering, payments, and cart abandonment. React to event triggers such as payment success/failure, chat completion, and cart abandonment by sending outbound HTTP requests, WhatsApp messages, Telegram messages, or emails. Supports lead generation, SMS campaigns, and integration with e-commerce platforms like Shopify and WooCommerce.
