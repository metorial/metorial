# Neutrino

Validate and clean email addresses and phone numbers, detecting disposable or invalid entries. Look up IP geolocation, ISP details, and reverse DNS. Check IP addresses and domains against blocklists and blacklists for security threats. Perform BIN/IIN lookups to identify card issuers and types. Geocode addresses to coordinates and reverse geocode coordinates to addresses. Send SMS messages and voice calls for two-factor authentication and security code verification. Convert currencies and measurement units. Render HTML to PDF, JPG, or PNG. Resize and watermark images. Generate QR codes. Filter bad words for content moderation. Parse user agent strings. Retrieve DNS and WHOIS information for domains. Detect VPN/proxy usage for IP addresses. Load and interact with web pages programmatically.
