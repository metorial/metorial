# <img src="logo.png" height="20"> Addresszen

Verify and standardize postal addresses, validate email addresses, and validate phone numbers. Autocomplete and search addresses in real-time with suggestions from authoritative postal datasets for US and international locations. Cleanse and verify full addresses against trusted sources to return standardized, deliverable address components. Manage API keys, retrieve usage statistics, and export lookup logs.
