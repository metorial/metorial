# <img src="logo.svg" height="20"> Bugbug

Run and manage automated end-to-end web application tests in the cloud. List and retrieve tests and suites within a project, trigger individual test runs or suite runs with custom variables and profile configurations, and retrieve test run results including pass/fail status. Export test reports in JUnit XML, JSON, and CSV formats. Useful for CI/CD pipelines and automated quality assurance workflows.
