# Ko Fi

Receive webhook notifications for payment events on Ko-fi, including donations, subscription payments, shop orders, and commissions. Each webhook payload includes supporter details, payment amount, currency, transaction type, and order-specific information such as shop items and shipping details. Verify webhook authenticity using a verification token. Note: Ko-fi is webhook-only (outbound) with no queryable API — you cannot read, create, or manage Ko-fi data directly.
