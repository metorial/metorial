Search, discover, and retrieve detailed information about places and venues from a global database of 100M+ points of interest across 200+ countries. Search places by keyword, category, chain, or geographic area with autocomplete support. Retrieve rich place details including addresses, coordinates, categories, hours, photos, reviews, ratings, and popularity. Match and enrich POI data using GeoTagging/Place Match. Propose edits and flag issues with place data. Manage geofences around venues, coordinates, or custom polygons with bulk operations. Provide personalized place recommendations based on user taste and visit history. Upload and manage geospatial datasets in Foursquare Studio. Access measurement data for campaign insights and optimization. Manage users and authentication tokens. Receive real-time webhook pushes for user check-ins and venue activity.
