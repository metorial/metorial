# Dictionary API

Look up word definitions, phonetics, pronunciations, synonyms, antonyms, and example sentences. Retrieve detailed linguistic information for words across multiple languages, including parts of speech, IPA transcriptions, and audio pronunciation URLs. Supports querying in any available language by language code.
