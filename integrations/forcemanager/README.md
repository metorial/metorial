# Forcemanager

Manage accounts, contacts, opportunities, activities, sales orders, and products in a mobile-first CRM for field sales teams. Create and update company records with geolocation and custom fields. Track sales activities including visits, calls, and meetings with check-in/check-out location data. Manage opportunities through their lifecycle with amount, probability, and status tracking. Create sales orders with line items, multi-level discounts, and product references. Maintain a product catalog with pricing and categorization. Manage calendar events and tasks linked to accounts, contacts, and opportunities. Search and filter records using advanced query capabilities. Map external system IDs for ERP/CRM synchronization.
