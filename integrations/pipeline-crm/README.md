# <img src="logo.png" height="20"> Pipeline Crm

Manage sales deals, contacts, companies, and activities in Pipeline CRM. Create, update, and search deals with associated values, stages, and currencies. Add and manage person and company records with duplicate detection and automatic company linking. Create calendar events and tasks associated with deals, people, or companies. Add categorized notes to CRM records. Read and write custom fields for companies, deals, and people. Capture leads from website forms via Web-to-Leads. Retrieve user information and assign owners to records.
