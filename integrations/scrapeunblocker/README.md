# ScrapeUnblocker

Scrape pages that block ordinary HTTP requests.

ScrapeUnblocker renders web pages in a real browser behind anti-bot protections
such as Cloudflare, DataDome, PerimeterX and Akamai, and returns raw HTML or
AI-parsed structured JSON. It also extracts Google search results.

## Authentication

API key. Get one at [scrapeunblocker.com](https://www.scrapeunblocker.com).

## Tools

| Tool | Description |
| --- | --- |
| `scrape_url` | Fetch a fully rendered page, bypassing anti-bot protection |
| `search_google` | Extract Google organic search results for a query |
