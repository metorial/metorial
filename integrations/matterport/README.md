# Matterport

Manage 3D digital twins of physical spaces on the Matterport platform. Search, create, update, and archive 3D models and their metadata including names, addresses, and visibility settings. Access and download model assets such as floor plans, point clouds, OBJ meshes, and panoramic imagery. Create, edit, and delete annotation tags (Mattertags) on 3D models. Purchase and manage add-on bundles like MatterPaks and floor plans. Manage folders, users, permissions, and custom roles at the account level. Import panoramic images for 3D reconstruction. Access AI-derived property intelligence including dimension estimates, room classification, and 2D floor plan geometry. Manage views and layers within models. Subscribe to webhook events for model lifecycle changes, state changes, transfers, and bundle updates.
