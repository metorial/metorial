# <img src="https://provider-logos.metorial-cdn.com/optimizely.png" height="20"> Optimizely

Manage A/B testing experiments, feature flags, and content across Optimizely's digital experience platform. Create and configure web experiments, feature rollouts, and flag rulesets with traffic allocation. Manage content marketing workflows including tasks, campaigns, assets, briefs, and publishing events. Upload and organize digital assets in folders with permissions. Send and track email marketing campaigns with recipient lists, target groups, and opt-in processes. Access real-time email engagement events including opens, clicks, bounces, and unsubscribes. Manage CMS content items, content types, and site definitions. Query indexed content via GraphQL with filtering, sorting, and faceting. Integrate customer data through ODP including profiles, events, segments, and real-time audience segmentation. Export raw experimentation event data for analysis. Receive webhook notifications for datafile updates, content workflow changes, email engagement events, and content synchronization.

## Tools

### Manage Audience

Create, update, retrieve, or list audiences in Optimizely Experimentation. Audiences define targeting criteria that can be applied to experiments to control which visitors are included.

### Manage CMP Asset

Retrieve, update, delete, or list assets in the Optimizely Content Marketing Platform library. Assets include images, videos, articles, raw files, and structured content. Use this to browse, search, and manage assets and folders.

### Manage CMP Campaign

Create, update, retrieve, or list marketing campaigns in Optimizely Content Marketing Platform. Campaigns are top-level containers for organizing related content tasks, briefs, and assets.

### Manage CMP Task

Create, update, retrieve, delete, or list tasks in Optimizely Content Marketing Platform. Tasks represent content work items that move through workflow stages. Use this to manage the full lifecycle of content tasks including commenting.

### Manage CMS Content

Create, update, retrieve, delete, publish, or list content items in Optimizely CMS (SaaS). Content items are managed as structured data with typed properties. Use this to manage the full content lifecycle including publishing.

### Manage ODP Customer

Retrieve or update customer profiles in Optimizely Data Platform (ODP). Use this to look up customer data by any identifier (email, customer ID, etc.) or upsert profile attributes. Also supports querying customer segments.

### Manage Experiment

Create, update, retrieve, or delete A/B test experiments in Optimizely Web or Feature Experimentation. Use this to set up new experiments with variations and metrics, modify existing experiments, or retrieve experiment configuration and results.

### Manage Feature Flag

Create, update, retrieve, delete, enable, or disable feature flags in Optimizely Feature Experimentation. Use this to manage feature flag lifecycle including toggling flags on/off per environment and configuring rulesets.

### Manage Mailing

Create, update, retrieve, send, copy, or list mailings in Optimizely Campaign. Mailings are email messages that can be designed, configured, and sent to recipient lists.

### Manage Project

Create, update, retrieve, or list Optimizely Experimentation projects. Projects are top-level containers that hold experiments, audiences, events, and feature flags.

### Manage Recipient List

Create, retrieve, or list recipient lists and manage recipients in Optimizely Campaign. Recipient lists hold subscriber data that mailings are sent to. Use this to manage lists, add or remove individual recipients.

### Send ODP Event

Send one or more customer events to Optimizely Data Platform (ODP). Events track customer actions and behaviors that feed into segmentation, analytics, and personalization. Also supports querying historical events for a customer.
