# Minerstat

Monitor and manage cryptocurrency mining operations for GPU and ASIC workers. Retrieve real-time and historical worker data including hashrate, temperature, power consumption, and estimated earnings. Remotely control workers with commands such as reboot, shutdown, restart, and start/stop mining. Create, update, and delete workers, groups, tags, and customer accounts. Manage GPU overclocking (ClockTune) profiles and worker configurations including mining client, profit switching, and electricity settings. Query coin profitability data including price, difficulty, rewards, and network hashrate, filterable by ticker or algorithm. Look up mining hardware specs with hashrate benchmarks and power draw for GPUs and ASICs. Browse a mining pools directory with fees, supported coins, and payout thresholds. Access monitoring statistics such as activity logs, balance tracking, group stats, and global stats. Receive webhook-based event notifications for triggers like GPU temperature alerts, worker offline events, hashrate drops, and error detection.
