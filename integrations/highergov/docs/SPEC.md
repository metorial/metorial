# Slates Specification for Highergov

## Overview

HigherGov is a market intelligence platform for U.S. government contracting and grants. It aggregates data from public and proprietary sources to provide searchable databases of federal and state/local contract opportunities, grant opportunities, contract awards, awardee profiles, government contacts, and M&A transactions. It also offers pursuit/pipeline management tools for business development teams.

## Authentication

HigherGov uses **API key** authentication. API keys are generated by account administrators from the HigherGov platform settings (gear icon → API, or at `https://www.highergov.com/api-management/`). The full key is only displayed once at creation time and must be securely saved.

For the main data API, the key is passed as a query parameter:

```
GET https://www.highergov.com/api-external/contract/?api_key=your-api-key-here&page_number=1
```

For the Zapier integration endpoints, the key is passed as an `X-API-KEY` header:

```
POST https://www.highergov.com/zapier/auth/
Headers: X-API-KEY: your-api-key-here
```

API access is available to all HigherGov subscribers. Access is invite-only and tied to an active subscription.

## Features

### Contract and Grant Award Lookup

Query historical federal prime contract awards (61M+), subcontract awards, IDV (Indefinite Delivery Vehicle) contracts, prime grant awards (4M+), and subgrant awards. Filter by modification date, search criteria, and other parameters. Useful for competitive analysis, market research, and identifying incumbents.

### Opportunity Search

Search federal contract opportunities (from SAM.gov and other sources), DLA opportunities, state and local (SLED) contract opportunities, pre-RFP/forecasted opportunities, grant opportunities, and SBIR opportunities. Supports filtering by source type, captured date, and saved search criteria via a `search_id` parameter that mirrors searches configured in the HigherGov web UI.

### Awardee Data

Access profiles for 1.5M+ government contractors and grant recipients, including parent/child subsidiary relationships. Useful for researching competitors, finding teaming partners, and analyzing market share.

### People and Contacts

Look up 130K+ government buyer contacts and contractor personnel records, supporting relationship-building and outreach efforts.

### Document Access

Retrieve 3M+ documents including contract and grant RFPs, federal schedules, and protest decisions. Document downloads are provided via time-limited URLs that expire after 60 minutes.

### Contract Vehicles

Query 4K+ contract vehicles (e.g., GSASchedules, GWACs, BPAs) for analysis and opportunity identification.

### Agencies and Programs

Access data on 3K+ government agencies and various program categories including defense programs, tech programs, grant programs, NAICS codes, and PSC codes.

### Pursuit Management

HigherGov Business Development users can access and manage their pursuit pipelines via the API. Pursuits can also be pushed to external CRMs via the Zapier integration.

### Quick Search

Lightweight search endpoints that power autocomplete and dropdown functionality, available for free to all subscribers. Useful for building search interfaces in external applications.

## Events

HigherGov supports webhook-based events through its **Zapier integration**. This requires a Standard or Leader plan subscription.

### Pursuit Added

When a user adds a pursuit to a Zapier-linked pipeline in HigherGov, a webhook fires sending opportunity details (title, description, source ID, dates, agency info, NAICS/PSC codes, contact information, set-aside type, and value) to a configured target URL. Pursuits can be added from Federal Contract Opportunities, State and Local Opportunities, Forecast Opportunities, DLA/DIBBS Opportunities, Grant Opportunities, IDV Awards, Prime Contract Awards, and Prime Grant Award pages. This is the only trigger type available and is designed to push opportunity data into external CRMs.
