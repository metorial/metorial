# <img src="logo.jpeg" height="20"> Mopinion

Collect, retrieve, and analyze user feedback across web, mobile, and email channels. Retrieve account information, manage deployments that control where feedback forms appear, and manage reports and datasets. Retrieve feedback data with filters for date, rating, NPS, CES, GCR, and tags. Access field definitions to understand feedback structure. Configure webhooks to trigger on new feedback submissions with conditional rules based on scores, answers, or device metadata.
