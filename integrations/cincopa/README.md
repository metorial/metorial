# <img src="logo.svg" height="20"> Cincopa

Manage multimedia galleries, assets, portals, and live streams. Upload, organize, search, and retrieve video, audio, and image assets. Create and customize media galleries with metadata, tags, and featured items. Set up portals as customizable landing pages or video hubs. Start, stop, and manage live video streams. Collect user-generated content via uploads and recordings. Track video analytics including viewer engagement heatmaps. Capture leads from video forms. Configure webhooks for real-time notifications on asset, gallery, account, lead, and analytics events.
