# <img src="logo.png" height="20"> Refiner

Manage customer feedback surveys for web and mobile applications. Create and retrieve survey responses (NPS, CSAT, CES, etc.), manage contacts and their traits, track user events, organize contacts into segments, and tag survey responses. Identify and update users with custom attributes, group users into accounts, and import user data from backend databases. Receive webhook notifications when surveys are completed, seen, or dismissed, when users enter segments, and when tags are added to responses.
