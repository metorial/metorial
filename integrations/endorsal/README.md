# <img src="logo.png" height="20"> Endorsal

Collect, manage, and display customer testimonials and reviews. Create and manage contacts for review request campaigns. Send automated review requests via email and SMS through AutoRequest campaigns. Create, retrieve, update, and delete testimonials with star ratings, tags, and customer details. Import reviews from external platforms like Google, Facebook, Yelp, and Tripadvisor. Generate pre-filled SuperLinks to reduce friction in the review process. List and monitor widget configurations for embedding social proof on websites. Track campaign delivery stats including open and click rates. React to new testimonial submissions via webhook events.
