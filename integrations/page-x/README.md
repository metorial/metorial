# Page X

Submit and capture leads into the PageXCRM platform from external sources such as websites, ad platforms, or other applications. Create new lead records with contact details including name, email, phone, source platform, and country. The API is a one-way lead ingestion mechanism designed to feed lead data into PageXCRM for sales management and tracking.
