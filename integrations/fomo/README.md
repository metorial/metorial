# <img src="logo.png" height="20"> Fomo

Manage social proof notifications that display real-time customer activity (purchases, signups, reviews) on websites. Create, retrieve, search, update, and delete notification events with details like customer name, location, and custom attributes. Manage notification templates that define display formats using variable placeholders. Retrieve application statistics and update display settings for how notifications appear to site visitors. Search events by specific fields and values. Access Fomo's public marketing metrics including signup counts, customer counts, and integrations.
