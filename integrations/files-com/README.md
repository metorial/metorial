# <img src="logo.png" height="20"> Files.com

Upload, download, list, copy, move, delete, and preview files and folders on a cloud-native managed file transfer platform. Manage users, groups, and folder-level permissions with role-based access control. Create and configure secure share links with password protection, expiration dates, and download limits. Connect remote servers including AWS S3, Azure, Google Cloud, SharePoint, Google Drive, Box, and Dropbox as mounts or syncs. Automate file operations (copy, move, delete, import, create folders) triggered by file events, schedules, or webhooks. Configure outbound webhooks to notify external systems of file activity filtered by event type and file patterns. Manage API keys, SSH keys, SSO configuration, site-level security policies, and audit logs for all file operations and API requests.
