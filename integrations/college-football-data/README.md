# College Football Data

Retrieve college football data including game results, scores, schedules, and play-by-play details. Access team and player statistics, both basic and advanced (SP+, Elo, SRS, FPI, EPA). Look up recruiting rankings, player ratings, and transfer portal entries. Query betting lines (spreads, over/unders, moneylines) from various sportsbooks. Retrieve weekly AP/Coaches poll rankings, win probabilities, predicted points, and adjusted metrics. Search player profiles, coaching records, NFL Draft picks, conference listings, and venue information. Access drive-level data, team rosters, talent composites, and historical matchup records.
