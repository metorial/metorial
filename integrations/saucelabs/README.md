# Saucelabs

Manage automated and manual testing of web and mobile applications across 700+ browser, OS, and device combinations. Create, monitor, and delete test jobs on virtual and real devices. Manage builds to group related test jobs. Upload mobile app binaries for distribution and testing. Run and manage API tests, performance tests, and visual snapshot comparisons. Control Sauce Connect proxy tunnels for testing behind firewalls. Manage accounts, teams, and user access keys. Query platform availability and supported browser/device combinations. Receive webhooks for test completions, visual build results, app feedback, and crash events.
