# Mxtoolbox

Run DNS and email authentication lookups for domains and IP addresses, including MX, SPF, DKIM, DMARC, BIMI, MTA-STS, PTR, and TXT records. Check domain blacklist status and diagnose email deliverability issues. Create, update, delete, and query infrastructure monitors for blacklist, SMTP, HTTP, TCP, DNS, and mail flow health. Manage monitor tags for organizing and filtering monitors. Set up short-term Quickwatch monitors that auto-expire. Track API usage against plan limits. Receive callback notifications when monitored resources change status.
