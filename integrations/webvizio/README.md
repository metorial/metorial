# <img src="logo.jpeg" height="20"> Webvizio

Create, find, update, and delete projects and tasks for visual bug tracking and website feedback. Manage comments and threaded discussions on tasks. Tasks capture visual annotations pinned to live UI elements along with automatic technical metadata including screenshots, console logs, network logs, DOM snapshots, and user action recordings. Generate AI-ready prompts and retrieve error logs, reproduction steps, and screenshots for tasks to assist AI coding tools. Synchronize projects, tasks, and comments with external systems using External IDs. Subscribe to and manage webhooks for project, task, and comment events.
