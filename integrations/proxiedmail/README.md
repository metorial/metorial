# <img src="logo.svg" height="20"> Proxiedmail

Create and manage proxy email aliases that forward incoming messages to real email addresses. List, update, and configure proxy emails with forwarding rules, webhook callbacks, and browsable settings. Browse and read received emails including subject, body, sender, headers, and attachments. Attach custom domains for creating proxy addresses on your own domain. Send messages and initiate conversations from proxy email addresses. Set up webhook receivers to get notified on incoming emails and poll for webhook payloads.
