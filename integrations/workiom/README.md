# <img src="logo.jpeg" height="20"> Workiom

Create, read, update, and delete records in custom no-code business applications. Manage apps and lists (tables), query records with filters and sorting, upload files, and retrieve list metadata including fields and views. Subscribe to webhooks for record creation and update events. Supports various field types including text, numbers, dates, linked records, users, files, and more.
