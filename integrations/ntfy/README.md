# Ntfy

Send push notifications to phones and desktops via HTTP-based pub-sub topics. Publish messages with titles, priorities, tags, emojis, markdown formatting, click actions, icons, and file attachments. Add interactive action buttons (view, HTTP, broadcast, copy) to notifications. Schedule delayed message delivery, update or delete previously sent notifications, and forward notifications to email or trigger phone calls. Subscribe to topics in real-time via JSON stream, SSE, raw, or WebSocket connections, with filtering and polling support. Manage access tokens and user accounts. Format incoming webhook payloads using message templates for services like GitHub, Grafana, and Alertmanager.
