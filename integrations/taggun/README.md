# <img src="logo.svg" height="20"> Taggun

Extract structured data from receipt and invoice images using OCR and machine learning. Submit documents as file uploads (JPG, PNG, PDF, GIF, HEIF), base64-encoded content, or URLs. Extract totals, tax amounts, dates, merchant details, and line items with confidence scores. Detect fraudulent, duplicate, tampered, or AI-generated receipts. Validate purchases against campaign rules for promotions, rebates, and warranty programs. Normalize merchant names, verify merchant locations, and categorize purchases by merchant and product type. Extract multi-tier tax details, verify VAT numbers, and read QR codes from receipts. Validate serial numbers and barcodes for warranty workflows.
