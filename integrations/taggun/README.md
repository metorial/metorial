# <img src="logo.svg" height="20"> Taggun

Extract structured data from receipt and invoice images using OCR and machine learning. Submit documents as base64-encoded content or public HTTPS URLs. Extract totals, tax amounts, dates, merchant details, line items, raw OCR text, and account-enabled fraud or categorization fields. Create and manage purchase validation campaigns, validate receipts against those campaigns, submit corrected feedback for model training, and manage product category files for purchase categorization.
