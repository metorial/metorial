# <img src="logo.png" height="20"> Telnyx

Cloud communications platform for programmable voice, messaging, and telephony. Make, receive, and control voice calls with actions like transfer, bridge, conference, record, play audio, gather DTMF, and text-to-speech. Send and receive SMS/MMS messages via long codes, toll-free, short codes, and alphanumeric sender IDs. Search, provision, port, and manage phone numbers globally. Perform two-factor authentication (Verify) via SMS, call, flashcall, WhatsApp, and PSD2. Look up number carrier and caller ID information. Send and receive faxes. Manage IoT SIM cards, data usage, and wireless connectivity. Create AI-powered chat completions, transcriptions, speech synthesis, and embeddings. Store files with S3-compatible object storage. Manage billing, invoices, usage reports, and account settings. Receive webhook events for calls, messages, fax, porting, and verification status updates.
