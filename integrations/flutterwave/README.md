# <img src="logo.png" height="20"> Flutterwave

Process payments across multiple African and global markets using cards, bank transfers, mobile money, USSD, and digital wallets. Send payouts to bank accounts, mobile money wallets, and Flutterwave wallets in multiple currencies. Create and manage subscription billing with payment plans. Generate virtual accounts (dynamic or static) for receiving bank transfer payments. Create and manage payout subaccounts (wallets) with wallet-to-wallet transfers. Pay bills including airtime, data, cable, power, and tolls. Manage transaction disputes, refunds, and settlements. Verify identity via bank account and BVN resolution. Receive webhook notifications for charge, transfer, and subscription events.
