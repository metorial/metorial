# <img src="logo.png" height="20"> Realphonevalidation

Validate US phone numbers and email addresses in real-time using live telco data. Check phone connection status (connected/disconnected), identify phone type (mobile, landline, VoIP), retrieve carrier and subscriber information, and look up Do Not Call list status including national, state, and DMA lists. Score phone numbers for fraud risk, detect reassigned numbers since a given consent date, and identify TCPA litigators. Verify email addresses for validity, disposable address detection, and active network presence. Supports full validation (Turbo), budget-friendly list scrubbing (Scrub), basic active number checks, and wireless/phone type identification for TCPA compliance.
