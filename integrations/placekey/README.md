# <img src="logo.png" height="20"> Placekey

Look up universal unique identifiers (Placekeys) for physical places worldwide using addresses or geographic coordinates. Match, deduplicate, and merge location data across datasets. Retrieve single or bulk Placekeys with optional response fields including confidence scores, normalized addresses, geocodes, parcel identifiers, and census geoids. Geocode addresses into latitude/longitude coordinates.
