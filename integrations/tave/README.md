# Tave

Manage leads, contacts, jobs, orders, and payments for photography and creative studio businesses. Create leads from external sources such as website forms. Create, search, and retrieve contacts with custom fields. Create and manage jobs with job types, stages, roles, event dates, and custom fields. Track orders and bookings including electronic client bookings. Monitor payments by brand and job type. Retrieve and manage brand configurations. Trigger automated workflows on contact creation, order booking, and payment events. Send outbound web requests from workflow automations to external services.
