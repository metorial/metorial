# <img src="logo.png" height="20"> Neuronwriter

Optimize content for SEO using semantic analysis and NLP-driven recommendations. List projects, create keyword queries, and retrieve SEO recommendations including target word counts, term suggestions for titles/headings/body, entity recommendations, content ideas, and competitor SERP data. Import and retrieve content (HTML or URL), evaluate content against SEO scoring criteria, and generate shareable collaboration URLs for editors and reviewers.
