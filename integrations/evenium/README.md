# <img src="logo.svg" height="20"> Evenium

Create and manage professional events with details like title, dates, location, and description. Add and track participants with registration status, contact details, and CRM mapping. Retrieve event lists and participant rosters. Manage contacts database. Monitor ticket orders and updates. Track participant status changes including invited, registered, confirmed, cancelled, and attended. Supports external ID fields for CRM integration with Salesforce and Veeva.
