# <img src="logo.png" height="20"> Membervault

Manage users, products, and engagement tracking for online courses, memberships, and digital products. Add or remove users from products, delete user accounts, and retrieve product listings. Monitor events including user signups, lesson and module completions, engagement point thresholds, hot lead detection, email consent changes, account creation, and first logins via webhooks.
