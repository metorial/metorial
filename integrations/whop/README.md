# Whop

Sell digital products, memberships, and subscriptions on the Whop platform. Create and manage products and pricing plans, accept payments via checkout links or embedded checkouts, and issue refunds. Manage memberships including canceling, pausing, resuming, and attaching metadata or software license keys. Create and manage promo codes with percentage or fixed-amount discounts. Automate payouts to users from ledger accounts. Handle payment disputes and resolution cases. Manage chat channels, DMs, forums, and notifications for community engagement. Create and manage courses, chapters, and lessons with student tracking. Manage leads, waitlist entries, and shipments. Access customer reviews. Build and distribute Whop apps across companies. Receive webhooks for payment, membership, entry, and setup intent events.
