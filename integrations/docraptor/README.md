# Doc Raptor

Convert HTML, CSS, and JavaScript content into PDF and Excel (XLS/XLSX) documents. Generate PDFs from inline HTML or URLs with advanced layout features including mixed page sizes, headers/footers, footnotes, table of contents, and page numbers. Create accessible PDFs meeting WCAG 2.0 and Section 508 standards. Convert HTML forms into PDF forms. Apply PDF security with encryption, password protection, and print/copy restrictions. Generate Excel spreadsheets from HTML/XML content. Create documents asynchronously for large or complex jobs with status polling and callback notifications. Host generated documents at shareable URLs with configurable download limits and expiration dates. List previously created documents and retrieve DocRaptor IP addresses for firewall whitelisting.
