# <img src="logo.png" height="20"> Ipinfo.io

Look up detailed information for any IPv4 or IPv6 address. Retrieve IP geolocation data (city, region, country, coordinates, timezone, postal code), ASN and network details, privacy/anonymity detection (VPN, proxy, Tor, relay, hosting), mobile carrier identification, company data, abuse contact information, and WHOIS records. Perform reverse IP lookups to find hosted domains, query IP ranges by domain, detect residential proxies, and execute bulk/batch lookups across multiple IPs in a single request.
