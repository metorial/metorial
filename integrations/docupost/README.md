# <img src="logo.png" height="20"> Docupost

Send physical postal mail programmatically, including letters and postcards, via U.S. mail. Provide recipient and sender addresses, supply content as a PDF URL or HTML, and send letters or postcards with specified images. Check account balance before sending. Cancel letters within one hour of submission. The service is HIPAA-compliant and supports sending checks via the dashboard.
