# <img src="logo.svg" height="20"> Gamma

Generate AI-powered presentations, documents, social media posts, and webpages from text prompts or existing templates. Control text handling modes (generate, condense, preserve), configure image sourcing (AI-generated, stock photos, GIFs, web images), and apply visual themes for colors and fonts. Export generated content as PDF or PPTX files. Share content with configurable access permissions and organize it into folders. Supports 60+ languages, customizable tone, audience targeting, card dimensions, and header/footer configuration.
