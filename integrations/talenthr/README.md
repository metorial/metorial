# Talenthr

Manage employee records, organizational structure, and time-off tracking in a cloud-based HRIS for small to medium businesses. Hire new employees, create departments, divisions, job titles, and locations. List absent employees, view upcoming absences, and approve or reject time-off requests. Manage job openings for applicant tracking and careers pages.
