# <img src="logo.jpeg" height="20"> Short Io

Create, manage, and track branded short links using custom domains. Shorten URLs individually or in bulk, configure custom slugs, expiration dates, password protection, and cloaking. Set up geo-targeting (country/region) and mobile-targeting (iOS/Android) redirects. Retrieve detailed click analytics including browser, location, referrer, and device breakdowns for individual links or entire domains. Generate QR codes for links, manage Open Graph metadata, manage multiple branded domains, and receive webhook notifications for link creation and click events.
