# <img src="logo.png" height="20"> Eventee

Manage events, sessions, and attendees on the Eventee event management platform. Retrieve and update event agendas, stages, tracks, and session details. Import, invite, and manage attendees with profile information. Handle attendee check-in status and RSVP responses. Manage speaker profiles and session assignments. Monitor event activity through webhooks for attendee joins, invitations, check-in changes, RSVP responses, and new registrations.
