# <img src="logo.svg" height="20"> Google Address Validation

Validate, standardize, and geocode postal addresses worldwide. Identify and correct individual address components, assess deliverability, and return formatted addresses with latitude/longitude coordinates and Google Place IDs. Determine whether an address is residential, commercial, or a PO Box. Enable USPS CASS™ processing for US and Puerto Rico addresses to obtain ZIP+4 codes, delivery point codes, and carrier route information. Provide validation feedback to improve future results.
