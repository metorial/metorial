# <img src="logo.png" height="20"> Graphhopper

Calculate optimal routes between multiple points with support for various vehicle profiles (car, bike, foot, truck). Solve vehicle routing and traveling salesman problems with real-world constraints like time windows, capacities, and driver skills. Compute distance and time matrices between origin and destination sets. Geocode addresses to coordinates and reverse geocode coordinates to place names. Generate isochrones and isodistances for reachability analysis. Snap GPS points to road networks via map matching. Cluster customers into groups for territory planning. Create and manage custom routing profiles to avoid, prefer, or exclude specific road types and areas.
