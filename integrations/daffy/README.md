# <img src="logo.png" height="20"> Daffy

Manage charitable donations through a donor-advised fund (DAF) platform. Create, list, and cancel donations to over 1.7 million U.S. nonprofits. Check fund balances, view contribution history, and track donation statuses. Search and look up nonprofits by name or EIN. Create and manage Daffy Gifts — digital charity gift cards for recipients. Retrieve user profiles, causes, and public donation activity.
