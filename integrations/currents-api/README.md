# Currents API

Search and retrieve global news articles from over 14,000 sources across 70 countries and 18 languages. Fetch latest news in real time, search historical articles by keywords, date range, language, country, category, and domain. Discover available news sources and filter by language or category. Analyze topic trends and sentiment over time. Access reference data for supported languages, regions, and categories.
