# Rentman

Manage rental projects, equipment inventory, crew scheduling, and financial documents for AV and event production. Create and update projects, subprojects, and project requests. Track equipment availability, serial numbers, stock movements, and repairs. Assign crew to projects, manage crew availability and rates, and track time registrations. Manage contacts and contact persons for CRM integration. Create and manage quotes, contracts, and invoices with line items. Handle subrentals from external suppliers. Plan vehicle assignments and appointments. Organize files and folders attached to projects or equipment. Receive webhooks for create, update, and delete events across all resource types.
