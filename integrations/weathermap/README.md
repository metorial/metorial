# Open Weather Map

Retrieve current weather conditions, forecasts, and historical weather data for any location worldwide. Access air quality indices and pollutant concentrations. Convert location names to coordinates and vice versa via geocoding. Get solar irradiance data and energy production estimates. Retrieve weather map tile layers for visual representation. Monitor road weather risk conditions. Receive severe weather alerts and push notifications. Access fire danger assessments and agricultural monitoring data. Supports queries by geographic coordinates, with configurable units, languages, and response formats.
