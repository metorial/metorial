# Humanitix

Retrieve event, order, and ticket information from the Humanitix not-for-profit ticketing platform. Fetch details about events owned by or shared with your account, query orders associated with specific events, inspect individual ticket details, and manage tags for categorizing events. A beta capability for creating and updating events is available by request. The API is primarily read-only with no webhook support.
