# <img src="logo.png" height="20"> Byteforms

Retrieve and manage forms, surveys, and quizzes built on the ByteForms platform. List forms associated with an account, access form metadata and configuration, and retrieve form submissions with cursor-based pagination. Export submission data in CSV or JSON formats. Supports polling for new submissions based on timestamps to detect new responses. Integrates with third-party services like Slack for forwarding submission data.
