# <img src="logo.svg" height="20"> Epic Games

Authenticate players via Epic Games accounts or external identity providers (Steam, PlayStation, Xbox, Nintendo) using OAuth 2.0. Manage cross-platform player identities, friends lists, and presence status. Track and record player stats, unlock achievements, and manage leaderboards. Create and manage game lobbies, sessions, and matchmaking. Store and retrieve player-specific cloud save data and game-level title storage. Verify game and DLC ownership, enumerate entitlements, and validate ownership tokens. Submit and query player reports, define and manage player sanctions (bans/suspensions). Access game analytics including player retention and online player counts.
