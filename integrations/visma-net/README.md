# <img src="https://cdn.prod.website-files.com/642284f33fc26c8faad516ef/691f1ae913577db612387a5f_Visma%20Net-logo_rgb_Ubuntu_Visma%20Net-logo-main-inline-black-Ubuntu.svg" height="20"> Visma Net

Connect to Visma Net ERP using the official Visma.net ERP API. List and inspect customers, suppliers, accounts, customer invoices, supplier invoices, projects, inventory items, and sales orders; download Visma attachment/blob content as Slate attachments; and check asynchronous background API operations.

This integration uses the interactive Visma Connect OAuth flow and requires a Visma Net tenant ID in configuration. The tenant ID is sent as `tenant_id` during Visma Connect token exchange and refresh. The upstream API title remains `Visma.net ERP API` in Visma's OpenAPI documents.
