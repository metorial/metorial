# <img src="logo.png" height="20"> Process Street

Create, manage, and run structured workflow runs (checklists) from templates. Complete, assign, and track tasks within workflows. Read and update form field values across workflow runs. Manage structured data set records (create, update, delete, list). Handle approval tasks (approve/reject). Set due dates, change workflow run status (active, complete, archived), and manage subtasks. Configure outgoing webhooks for workflow, form, and data set events. Trigger new workflow runs via incoming webhooks.
