# Encodian

Generate, manipulate, and convert documents across Word, PowerPoint, Excel, and PDF formats. Populate templates with dynamic data, merge and split PDFs, add watermarks, headers, footers, and page numbers. Convert between file formats including PDF, Word, Excel, HTML, images, and TIFF. Extract text from scanned documents and images using OCR. Extract structured data from invoices, receipts, IDs, contracts, and other document types using AI. Process images with cropping, compression, format conversion, and watermarking. Generate and scan barcodes. Automate Excel operations including find/replace and row updates. Perform text and security utilities such as AES/RSA encryption, hash generation, JWT creation, and regex operations. Create and extract ZIP archives. Process email files. Convert speech audio to text and translate files between languages.
