# <img src="logo.png" height="20"> Screenshot One

Capture website screenshots, render HTML/Markdown to images or PDFs, and extract page metadata. Supports full-page and element-level screenshot capture in multiple formats (PNG, JPEG, WebP, GIF, PDF, and more). Emulate devices and viewports, block ads/cookie banners/chat widgets, inject custom JavaScript/CSS, and customize rendering with dark mode, geolocation, and proxy-based geo-targeting. Generate PDFs with configurable paper size, orientation, and margins. Extract metadata including Open Graph tags, fonts, favicons, and page content. Upload screenshots directly to S3-compatible storage. Execute requests asynchronously with webhook delivery. Integrate with OpenAI Vision for prompt completions on captured screenshots. Track API usage programmatically.
