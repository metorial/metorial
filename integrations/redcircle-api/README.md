# Red Circle API

Search and retrieve product data from Target (US retailer). Look up products by TCIN, DPCI, UPC, GTIN, ISBN, or URL. Search Target's inventory by product name, SKU, or UPC with sorting and filtering. Retrieve customer reviews, browse category listings, and localize results by US zip code for shipping and availability. Run bulk collections of up to 15,000 requests on a schedule, export results as JSON or CSV, and automatically upload to Amazon S3 or Google Cloud Storage. Receive webhook notifications when collection runs complete.
