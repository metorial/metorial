# Hex

Manage and orchestrate data projects in a collaborative Hex workspace. List, create, and retrieve projects. Trigger and monitor runs of published projects with custom input parameters. Manage project sharing and access controls for users, groups, and collections. Generate presigned URLs for embedding Hex apps externally. Create, list, and manage users, groups, and collections. Configure data connections to external databases (Snowflake, BigQuery, Databricks, PostgreSQL, etc.) with schema filters and sharing permissions. Manage cells (SQL and code) within project drafts. Create, update, and publish guides. Ingest and manage semantic models. Retrieve observability data on queried tables.
