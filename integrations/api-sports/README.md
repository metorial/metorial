# <img src="logo.png" height="20"> API Sports

Retrieve real-time and historical sports data across 12 sports including football, basketball, NBA, NFL, baseball, hockey, Formula 1, MMA, rugby, handball, volleyball, and AFL. Look up leagues, teams, players, fixtures, and live scores updated every 15 seconds. Access match events (goals, cards, substitutions), lineups, formations, and detailed match and season statistics. Retrieve league standings, pre-match and live betting odds, AI-generated match predictions, player injuries, coach profiles, and transfer history. Query Formula 1 race results, pit stops, and driver rankings. Filter data by league, season, date, team, round, or status.
