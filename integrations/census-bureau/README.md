# <img src="logo.png" height="20"> Census Bureau

Query raw statistical data from U.S. Census Bureau programs, including demographic, economic, housing, and geographic datasets. Retrieve population, income, housing characteristics, and other variables filtered by geographic area (state, county, tract, block group, etc.) and vintage year. Discover available datasets, variables, and supported geographies. Geocode addresses to latitude/longitude coordinates and identify Census geographic areas. Access geographic boundary shapes via TIGERweb. Query microdata for individual-level survey responses. Retrieve international demographic indicators for countries worldwide. Access economic time series data covering construction, trade, manufacturing, and more.
