# Chatbotkit

Build, manage, and deploy AI chatbots and conversational agents. Create and configure bots with custom backstories, AI models (OpenAI, Anthropic, etc.), datasets, and skillsets. Manage conversations with streaming responses, function calling, and context-aware memory. Import knowledge from web pages, PDFs, DOCX, and Notion into datasets. Define skillsets with custom abilities for actions like web fetching and email sending. Track and manage contacts with personalized experiences and scheduled actions. Deploy bots to Slack, Discord, WhatsApp, Microsoft Teams, Google Chat, Twilio SMS, and web widgets. Organize resources into spaces and blueprints for team collaboration. Extract structured data from conversations using custom schemas. Manage files, secrets, audit logs, and content moderation settings. Support multi-tenancy via the Partner API with sub-account creation and impersonation. Receive real-time webhook notifications for bot, conversation, and integration events.
