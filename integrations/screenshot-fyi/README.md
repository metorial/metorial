# <img src="logo.png" height="20"> Screenshot.fyi

Capture high-quality screenshots of any website via API. Supports full-page captures, custom viewport sizes (mobile to widescreen), dark mode, automatic cookie banner and popup removal, and multiple output formats (PNG, JPG). Returns a URL to the captured screenshot image.
