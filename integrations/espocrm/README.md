# <img src="logo.svg" height="20"> Espocrm

Manage contacts, accounts, leads, opportunities, and support cases in a self-hosted CRM. Create, read, update, and delete records across all entity types. Track sales pipelines and convert leads into contacts and opportunities. Manage meetings, calls, tasks, and calendar events linked to CRM records. Send emails, manage email templates, and run mass email campaigns. Upload file attachments, generate PDF documents from templates, and import data from CSV. Link related records together, post to activity streams, and manage custom entities and fields. Subscribe to webhooks for record creation, update, deletion, and field change events.
