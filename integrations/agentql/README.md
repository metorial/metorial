# <img src="logo.png" height="20"> Agentql

Extract structured data from web pages, PDFs, and images using natural language or structured queries. Query any public URL or raw HTML to retrieve JSON-formatted data, with support for dynamic content, authenticated pages, and cross-site compatibility. Upload PDF and image files to extract structured information. Create remote browser sessions with CDP access for browser automation, including proxy configuration and stealth mode for bot detection avoidance.
