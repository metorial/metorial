# <img src="logo.png" height="20"> Heyzine

Convert PDF, DOCX, and PPTX files into interactive digital flipbooks with page-turning effects. Create flipbooks programmatically from document URLs with customizable styling, titles, logos, and UI controls. List, retrieve details, and delete flipbooks. Organize flipbooks into bookshelves by adding, removing, and reordering them. Configure social media metadata (title, description, thumbnail) for flipbooks and bookshelves. Protect flipbooks with password-based access control including per-user credentials, single passwords, Google Sign-In, OTP, and email-based access. Retrieve oEmbed data for embedding flipbooks in third-party platforms. Collect leads via webhook when users submit forms embedded in flipbooks.
