# <img src="logo.png" height="20"> Mails.so

Validate and verify email addresses for deliverability and quality. Check individual emails or bulk-validate lists for format validity, domain and MX record status, catch-all policies, disposable address detection, and blocklist presence. Returns deliverability scores and classifications to help maintain clean email lists.
