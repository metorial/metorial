# Smartproxy

Scrape and extract structured data from websites using 100+ pre-built templates for eCommerce, search engines, and social media platforms. Route web requests through residential, mobile, ISP, and datacenter proxy pools across 195+ geo-locations. Manage proxy users, whitelisted IPs, and traffic limits via the Public API. Supports real-time and asynchronous scraping with configurable JavaScript rendering, CAPTCHA solving, device emulation, and output parsing (JSON, HTML, Markdown, screenshots). Schedule recurring scrape jobs with delivery via email, webhook, or Google Drive. Track traffic usage, request success rates, and spend across all proxy products. Receive webhook notifications for payment failures, usage threshold alerts, subscription events, and account status changes.
