# Tldv

Retrieve recorded meeting data, transcripts, and AI-generated highlights from Zoom, Google Meet, and Microsoft Teams calls. Download meeting recordings via signed URLs, access structured transcripts with speaker identification and timestamps, and retrieve AI-generated notes organized by topic. Import external recordings for transcription and analysis. Receive webhook notifications when meetings finish processing or transcripts become available.
