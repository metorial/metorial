# <img src="logo.png" height="20"> Ip 2 Proxy

Detect whether an IP address is associated with a proxy, VPN, TOR exit node, data center, search engine robot, or residential proxy. Query IPv4 or IPv6 addresses to retrieve proxy type, geolocation data (country, region, city), network metadata (ISP, domain, ASN), threat classifications (spam, scanner, botnet), and VPN provider identification. Check remaining API credit balance. Supports multiple response packages (PX1–PX11) with varying levels of detail.
