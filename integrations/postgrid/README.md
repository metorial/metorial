# <img src="logo.jpeg" height="20"> Postgrid

Send physical mail programmatically including letters, postcards, cheques, and self-mailers. Manage contacts and reusable templates with merge variables for personalized mailings. Verify, standardize, and autocomplete postal addresses across the US, Canada, and 245+ countries. Track mail order status through printing, delivery, and completion stages. Run bulk mail campaigns to contact lists. Process PDFs with OCR address extraction and HTML-to-PDF rendering. Add QR codes and personalized URLs to track mailpiece engagement. Manage return envelopes and detect recipient address changes via NCOA. Receive webhook notifications for mail order lifecycle events.
