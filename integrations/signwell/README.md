# Signwell

Create, send, and manage electronic signature documents. Upload files, define signature fields, configure multi-signer workflows, and send documents for legally binding e-signatures. Create reusable templates with pre-defined fields and generate documents from them with dynamic recipient data. Support embedded signing experiences within third-party websites via iFrame. Bulk send template-based documents to many recipients using CSV data. Manage document lifecycle including sending reminders, updating recipients, downloading signed PDFs, and tracking status. Configure webhooks for real-time notifications on document events such as created, sent, viewed, signed, completed, declined, and expired.
