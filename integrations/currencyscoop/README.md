# Currency Scoop

Retrieve real-time and historical exchange rates for 168 fiat currencies and 2,000+ cryptocurrencies. Convert amounts between currencies using live mid-market rates. Fetch historical rates for any date back to 1996. Query time-series exchange rate data between date ranges for charting and volatility analysis. List all supported currencies with names, symbols, and ISO codes.
