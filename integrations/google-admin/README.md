# <img src="logo.svg" height="20"> Google Admin

Manage Google Workspace organizations programmatically. Create, update, delete, and search users, groups, and organizational units. Manage user aliases, group memberships, and group settings. Assign and revoke admin roles and privileges. Manage ChromeOS and mobile devices including remote actions like wipe and deprovision. Add and manage domains and domain aliases. Assign, update, and revoke product licenses. Access audit logs and usage reports for admin actions, logins, Drive activity, and device events. Monitor and respond to security alerts for phishing, malware, and suspicious activity. Transfer data ownership between users. Manage calendar resources such as meeting rooms and equipment. Configure SSO and email routing settings. Manage Chrome printers and shared contacts. Subscribe to webhook notifications for user changes and activity events.
