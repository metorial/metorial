# Engage

Manage customers, track events, and send transactional messages across multiple channels. Create, update, archive, and merge user profiles with custom attributes. Organize customers into accounts and subscription lists with double opt-in support. Track custom events for segmentation and automation. Send transactional emails using HTML, text, or templates with tracking options. Send transactional SMS via integrated providers like Twilio and Termii. Manage device tokens for push notifications. Perform batch operations for bulk user and event processing.
