# <img src="logo.png" height="20"> Sendloop

Manage email marketing campaigns, subscriber lists, and transactional emails. Create and send one-time or scheduled email campaigns with A/B testing. Add, update, search, and segment subscribers across lists with custom attributes. Track campaign performance with real-time reporting on opens, clicks, bounces, unsubscribes, and forwards. Send transactional emails for signups, password resets, and order confirmations. Set up automated email workflows triggered by customer behaviors. Configure RSS-to-email campaigns and manage sub-user accounts with granular permissions. Receive webhooks for subscription, unsubscription, hard bounce, and spam complaint events.
