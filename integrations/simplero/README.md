# Simplero

Manage contacts, mailing lists, products, purchases, invoices, and account administrators for an all-in-one online course and membership platform. Create and update contacts with custom fields, tags, GDPR consent, and affiliate attribution. Subscribe and unsubscribe contacts to mailing lists individually or in bulk. List products, look up purchases, and add people to products via webhooks. Retrieve invoices filtered by date or number range. Manage account administrators with role assignments. Receive webhooks for events including list subscriptions, purchases, tag changes, course completions, membership joins, form submissions, event registrations, pipeline updates, email clicks, and affiliate activity.
