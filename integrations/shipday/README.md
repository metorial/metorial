# Shipday

Create, retrieve, edit, and delete delivery and pickup orders for local businesses. Manage carrier/driver fleets by adding, removing, and assigning drivers to orders. Track delivery progress with real-time ETA, distance, and driver location. Connect to third-party on-demand delivery services (DoorDash, Uber) to check availability, get cost estimates, assign deliveries, and cancel assignments. Receive webhook notifications for order status changes and driver location updates. Access partner-level APIs to query merchant members and their completed orders.
