# <img src="logo.png" height="20"> Affinda

Extract structured data from documents using AI-powered parsing. Upload resumes, invoices, bank statements, passports, job descriptions, and other documents (PDF, DOCX, JPG, PNG, etc.) to receive clean structured JSON output. Classify documents automatically by type, validate extracted data with configurable rules, and redact sensitive information from resumes. Match resumes against job descriptions to rank candidates or jobs. Manage workspaces, collections, organizations, and document annotations. Receive webhook notifications when documents are parsed or validated.
