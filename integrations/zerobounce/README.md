# <img src="logo.jpeg" height="20"> Zerobounce

Validate email addresses for deliverability, detecting invalid, catch-all, spam trap, abuse, and disposable emails. Score email quality using AI to assess subscriber engagement likelihood. Find business email addresses by name and domain. Search domains for email format patterns. Check email activity data (opens, clicks, forwards, unsubscribes) to identify active inboxes. Upload and process bulk email lists for validation and scoring. Evaluate email list health with estimated bounce rates. Retrieve WHOIS and website availability information for domains. Manage allowlists and blocklists for custom validation filtering. Check account credit balance and API usage statistics.
