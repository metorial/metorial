# <img src="logo.png" height="20"> Instacart

Create shoppable recipe pages and shopping lists with ingredient-to-product matching on Instacart Marketplace. Discover nearby retailers by postal code. Manage grocery delivery and pickup order fulfillment including creating, updating, and canceling orders. Query available stores, delivery time slots, and pickup windows. Track order status through lifecycle events (picking, delivering, delivered, canceled). Receive real-time item-level updates (found, replaced, refunded) and delivery tracking with shopper GPS coordinates. Serve and track Carrot Ads (sponsored products, banner ads, brand pages) across storefronts. Ingest and manage product catalog data. Handle post-checkout order status pages. Collect order feedback and ratings. Manage customer data for privacy compliance. Link retailer customer accounts to Instacart accounts. Simulate order workflows in a sandbox environment.
