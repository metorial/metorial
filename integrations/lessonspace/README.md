# <img src="logo.svg" height="20"> Lessonspace

Create and manage virtual classroom spaces for online tutoring and teaching. Launch spaces with configurable whiteboards, video conferencing, and document editing. Generate per-user join URLs with customizable roles, permissions, and feature toggles. Manage sessions, retrieve session recordings, transcriptions, and AI-generated lesson summaries. List and manage organisation users. Configure space branding, theming, locale, and access restrictions. Receive real-time webhooks for session, user, chat, co-browsing, transcription, and summary events.
