# <img src="logo.jpeg" height="20"> Interzoid

Match, deduplicate, standardize, enrich, and validate data using AI-powered APIs. Generate similarity keys for fuzzy matching of company names, individual names, addresses, and product names. Compare two values and receive match scores indicating likelihood of being the same entity. Enrich records with business information, executive profiles, stock analysis, and custom structured data. Standardize company names, country names, state/province names, and city names to canonical forms. Classify data by entity type, gender, name origin, and language. Validate email addresses with deliverability and demographic data. Parse addresses into structured components. Process entire datasets and files in batch via CSV/TSV or cloud database connections. Retrieve live currency exchange rates, weather data, and global page load performance metrics.
