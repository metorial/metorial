# Slates Specification for Interzoid

## Overview

Interzoid is an AI-powered data quality platform that provides REST/JSON APIs for data matching, data standardization, data enrichment, and data validation. Interzoid's APIs allow you to match records, enrich attributes, normalize and standardize fields, and verify data across multiple domains. It offers 30+ APIs accessible via a single API key.

## Authentication

Interzoid uses API key authentication. All Interzoid APIs can be accessed using a single API license key, which is provided upon registration.

- **API Key**: The API key is a parameter that is used with each call to the API to identify your account and track usage.
- **How to obtain**: Register using your email address, verify your email, and your account will be provisioned with an API license key for access that you can obtain after logging in.
- **Usage in requests**: The API key is passed as a query parameter named `license` in each API call. For example:
  ```
  https://api.interzoid.com/getcompanymatchadvanced?company=IBM&algorithm=ai-plus&license=YOUR_API_KEY
  ```
- The API key will be created with a number of starting credits, enabling immediate, unrestricted use of the APIs.

## Features

### Data Matching (Similarity Keys)

Generate AI-powered similarity keys for data records so that similar values (e.g., "IBM" and "International Business Machines") produce the same key, enabling deduplication and fuzzy matching. A Similarity Key is a hash generated by Interzoid's algorithms using AI Models, knowledge bases, algorithms, and various heuristic methods to identify and cluster similar data elements. Data that is similar will generate the same Similarity Key.

- Supported categories: company/organization names, individual full names, US street addresses, global addresses, and product names.
- An `algorithm` parameter can be specified (e.g., `ai-plus`) for enhanced matching on some endpoints.

### Match Scoring

Compare two values and receive a match score (0–100) indicating the likelihood that they refer to the same entity. This API provides a match score (likelihood of matching) between two individual names on a scale of 0-100, where 100 is the highest possible match.

- Available for organization names and individual full names.

### AI-Powered Data Enrichment

Define exactly what data you need, and the AI will retrieve and structure it for you. The custom data enrichment API accepts a topic, a lookup value, and a list of desired output fields (e.g., headquarters, CEO, revenue) and returns structured JSON.

- Additional specialized enrichment APIs provide: business information, parent company info, executive profiles, email trust scores, IP address profiles, phone number profiles, company/organization verification, stock analysis by ticker, and X/Twitter handle/profile data.
- These are premium APIs that consume multiple credits per call.

### Data Standardization and Normalization

Standardize data values to consistent, canonical forms for improved data consistency and analysis. Available standardization categories include:

- Company/organization names
- Country names (with optional additional country information)
- State/province names (including abbreviations)
- City names

### Data Analysis and Classification

Use AI to analyze and classify data values in real time:

- **Entity type detection**: Determine whether a data value is a name, company, location, etc.
- **Gender determination**: Infer gender from individual name data.
- **Name origin detection**: Determine the cultural/ethnic origin of individual names.
- **Language identification**: Detect the language of a text input.
- **Translation**: Detect language and translate text to English or any other specified language.

### Email Validation

The API provides validation information for email addresses to aid in deliverability and to prevent sending email to bad addresses. It also offers additional demographic and descriptive data useful for marketing, personalization and segmentation purposes.

### Address Parsing

Parse addresses into their component parts (street, city, state, zip, etc.) for structured data handling.

### Utility APIs

- **Global page/API load performance**: Measure page load times or API response times from 20+ global locations.
- **Live currency exchange rates**: Retrieve real-time exchange rates for 150+ global currencies.
- **Weather information**: Get current weather data by US zip code or global location.
- **Remaining credits**: Check remaining API credit balance for your account.

### Batch / Full Dataset Processing

Process entire data tables, datasets, and files with a single API call. Generate match reports, append similarity keys to tables and files, append and enrich entire datasets with new columns, and more.

- Supports CSV/TSV files, and connections to cloud databases (Snowflake, Postgres, MySQL, Azure SQL, AWS RDS, etc.).
- The `connect.interzoid.com` endpoint accepts parameters for function, source, connection, table, and column to run batch jobs.

## Events

The provider does not support events. Interzoid is a stateless, request-response API platform with no webhook or event subscription mechanism.
